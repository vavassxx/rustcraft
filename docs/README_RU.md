# Документация RustCraft

Добро пожаловать в документацию RustCraft. RustCraft — это мод-утилита для Minecraft Fabric, который позволяет загружать моды, написанные на Rust.

## Быстрый старт

### Для игроков

1. Установите Fabric Loader для вашей версии Minecraft
2. Скачайте JAR загрузчика RustCraft для вашей версии
3. Поместите его в папку `.minecraft/mods`
4. Скачайте JAR-обёртки модов на Rust и поместите их в папку mods
5. Запустите Minecraft

### Для разработчиков

1. Клонируйте репозиторий
2. Соберите Rust SDK: `cd rust-sdk && cargo build --release -p rustcraft-core`
3. Соберите загрузчик: `cd fabric-loader && ./gradlew build`
4. Создайте свой мод, используя пример как шаблон
5. Соберите и установите свой мод

## Структура проекта

```
rustcraft/
├── fabric-loader/          # Java Fabric мод (загрузчик)
│   ├── src/main/java/com/rustcraft/
│   │   ├── RustCraftMod.java
│   │   ├── RustModLoader.java
│   │   ├── RustNativeBridge.java
│   │   └── mod/
│   └── src/main/resources/
├── rust-sdk/              # Rust SDK
│   ├── rustcraft-core/    # Ядро нативной библиотеки
│   └── rustcraft-api/     # API для разработчиков модов
├── example-mod/           # Пример мода
│   ├── src/lib.rs         # Исходный код мода на Rust
│   └── wrapper/           # Java-обёртка для загрузки через Fabric
└── docs/                  # Документация
```

## Ключевые возможности

- **Поддержка Rust**: Пишите моды для Minecraft на Rust
- **Мноверсионность**: Поддержка Minecraft 1.18.2–1.21.5
- **Кроссплатформенность**: Работает на Windows, Linux и macOS
- **Безопасность**: Безопасность памяти Rust предотвращает распространённые ошибки
- **Производительность**: Нативный код для критичных к производительности операций
- **Совместимость**: Работает с популярными модами Fabric

## Архитектура

RustCraft состоит из двух основных компонентов:

1. **Java Fabric мод**: Обеспечивает интеграцию с Fabric и обнаружение модов
2. **Rust нативная библиотека**: Предоставляет среду выполнения для модов на Rust

Моды на Rust загружаются через Java-обёртки. Каждый Rust-мод поставляется вместе с Java-классом, реализующим интерфейс `RustModWrapper`. Fabric обнаруживает обёртки через `fabric.mod.json`, вызывает их `onInitialize()`, после чего обёртка извлекает нативную библиотеку из своего JAR и регистрируется в `RustModLoader`.
