# Руководство по совместимости RustCraft

Этот документ описывает информацию о совместимости RustCraft с различными версиями Minecraft и популярными модами Fabric.

## Совместимость с версиями Minecraft

### Поддерживаемые версии

RustCraft поддерживает версии Minecraft от 1.18.2 до 1.21.5:

- **1.18.2** — Полная поддержка
- **1.19.4** — Полная поддержка
- **1.20.1** — Полная поддержка
- **1.20.4** — Полная поддержка
- **1.21.5** — Полная поддержка

### Сборка для всех поддерживаемых версий

```bash
./run-client.sh --build-all
```

## Конфликты нативных библиотек

### Конфликты версий библиотек

RustCraft загружает нативные библиотеки изолированно:

- Каждый мод получает собственный экземпляр нативной библиотеки
- Глобальные символы не экспортируются
- Конфликты версий предотвращаются

### Проблемы для конкретных платформ

#### Linux

- Некоторые дистрибутивы могут блокировать загрузку библиотек
- Решение: Убедитесь в правильных разрешениях
- Может потребоваться настройка SELinux

#### Windows

- Антивирус может помечать нативные библиотеки
- Решение: Добавьте исключения для каталога Minecraft
- SmartScreen может предупреждать при первом запуске

#### macOS

- Gatekeeper может блокировать неподписанные библиотеки
- Решение: Разрешите неподписанные библиотеки в Системных настройках
- Apple Silicon требует сборки для ARM64

## Влияние на производительность

RustCraft оказывает минимальное влияние на производительность:

- Overhead нативного моста: < 1 мс на вызов
- Overhead памяти: ~5 МБ базовый + выделение на мод
- Время запуска: +100–200 мс для нативной инициализации

## Конфликты ресурсов

RustCraft не использует ресурсы Minecraft напрямую:

- Нет конфликтов с пакетами ресурсов
- Нет конфликтов с ассетами
- Нет конфликтов пространств имён (использует пространство имён "rustcraft:")

## Сетевая совместимость

Моды RustCraft по умолчанию являются только клиентскими:

- Нет требований к серверу
- Нет модификаций сетевого протокола
- Совместимость с ванильными серверами

## Устранение неполадок совместимости

### Мод не загружается

1. Проверьте логи на наличие конкретных сообщений об ошибках
2. Убедитесь, что мод совместим с вашей версией Minecraft
3. Убедитесь, что не установлены конфликтующие моды
4. Попробуйте загрузить RustCraft сначала один

### Ошибки нативных библиотек

1. Убедитесь, что нативная библиотека собрана для вашей платформы
2. Проверьте, что архитектура библиотеки совпадает с JVM (x64 vs x86)
3. Убедитесь в правильных разрешениях на файлы
4. Временно отключите программу обеспечения безопасности

### Проблемы производительности

1. Проверьте, установлены ли другие моды для повышения производительности
2. Убедитесь, что аргументы JVM подходящие
3. Мониторьте использование памяти
4. Проверьте конфликты модов

## Тестирование совместимости

Для тестирования совместимости с новым модом:

1. Установите RustCraft один и убедитесь, что он работает
2. Добавьте интересующий мод
3. Протестируйте основную функциональность
4. Проверьте логи на наличие предупреждений или ошибок
5. Сообщите о найденных проблемах
